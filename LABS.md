# LABS

## Lab Scenarios

### Scenario 1: Network Configuration
- Objective: Set up a basic network configuration using Cisco Packet Tracer.
- Steps:
  1. Create a new project in Packet Tracer.
  2. Add routers and switches as per the lab diagram.
  3. Configure the routers with appropriate IP addresses.

### Scenario 2: Web Server Setup
- Objective: Install and configure a web server using Apache.
- Steps:
  1. Launch a virtual machine with Ubuntu server.
  2. Update the package manager.
  3. Install Apache with `sudo apt-get install apache2`.

## Hands-On Exercises

### Exercise 1: SSH Configuration
- Task: Configure SSH on a Linux server.
- Steps:
  1. Access the Linux server via console.
  2. Install OpenSSH server.
  3. Enable and start the SSH service.

### Exercise 2: Firewall Rules
- Task: Set up firewall rules using `iptables`.
- Steps:
  1. Check current rules with `iptables -L`.
  2. Add rules to allow HTTP traffic.
